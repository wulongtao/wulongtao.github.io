OpenClaw+本地大模型搭建

2026-03-15
AI
536 词
  • AI
  • OpenClaw
  1. 准备工作：环境搭建
    1. 1. 安装 Node.js (运行环境)
    2. 2. 下载 Llama.cpp (推理引擎)
    3. 3. 从魔塔社区下载 Qwen 模型
  2. 安装 OpenClaw
  3. 配置好启动：openclaw gateway --port 18789 --verbose

本教程将手把手教你如何在本地环境下，利用 OpenClaw 作为界面，Llama.cpp 作为引擎，并从国内 魔塔社区 下载通义千问（Qwen）模型。

准备工作：环境搭建

在开始前，请按顺序安装以下三个核心组件。

1. 安装 Node.js (运行环境)

  • 访问地址：Node.js 官网 (nodejs.org)
  • 下载步骤：点击页面上的 LTS (Long Term Support) 版本下载并安装。
  • 验证：打开终端（CMD 或 PowerShell），输入 node -v，看到版本号即安装成功。

2. 下载 Llama.cpp (推理引擎)

  • 访问地址：GitHub Releases (llama.cpp)
  • 下载步骤：
    1. 找到最新版本（Latest）。
    2. 下载Windows x64 (CUDA 13) 和 CUDA 13.1 DLLs（具体看显卡版本的要求）
    3. 将2个压缩包解压到同一个文件夹，如E:\AI\llama-b8353-bin-win-cuda-13.1-x64

3. 从魔塔社区下载 Qwen 模型

由于模型文件较大，推荐使用国内高速的魔塔社区。

  • 访问地址：Qwen3.5-9B-GGUF
  • 下载步骤：
    1. 在文件列表中找到 Qwen3.5-9B-IQ4_NL.gguf。
    2. 点击旁边的下载按钮。
    3. 建议将该文件移动到你的 E:\AI\llama_models 文件夹下。

安装 OpenClaw

OpenClaw 是一个精美的网页交互界面，支持多种模型后端。

打开终端，运行以下命令全局安装：

npm install -g openclaw

提示：如果安装进度条不动，可以使用国内镜像：
npm install -g openclaw –registry=https://registry.npmmirror.com

启动流程（共两步）
第一步：启动 Llama.cpp 服务端
首先，我们需要让模型“跑起来”。进入 D:\AI-Tool 目录，打开 PowerShell 运行：

# -m 后面跟的是你模型文件的实际路径
 .\llama-server.exe -m E:\AI\llama_models\Qwen3.5-9B-IQ4_NL.gguf -ngl 50 -c 32000 -fa on --port 11434  -t 12

注意：看到 HTTP server listening 提示后，不要关闭这个窗口。

第二步：启动 OpenClaw 界面
打开一个新的终端窗口，输入：

openclaw onboard

配置参数如下：

Provider: 选择 Custom

API Base URL: http://localhost:11434

API Key: 随便填写（例如：sk-admin）

Model Name: 填写 http://127.0.0.1:11434/models 地址中获取到的name（Qwen3.5-9B-IQ4_NL.gguf）

配置好启动：
openclaw gateway --port 18789 --verbose
